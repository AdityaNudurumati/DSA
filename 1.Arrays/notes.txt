What is the Partition / Dutch Flag pattern?
The goal is to rearrange elements in-place into groups based on some condition.
Typical examples:
| Problem                             | Pattern             |
| ----------------------------------- | ------------------- |
| Sort Colors                         | Dutch National Flag |
| Move Negatives to Left              | Partition           |
| Segregate Even/Odd                  | Partition           |
| Partition Array Around Pivot        | Partition           |
| Quick Sort Partition Step           | Partition           |
| Quick Select (Kth Largest/Smallest) | Partition           |

Recommended Problems:
Easy
- Sort Colors
- Move Zeroes
- Segregate Even and Odd
- Move Negative Numbers to One Side

Medium
- Partition Array Around Pivot
- Sort Array By Parity
- Wiggle Sort
- Kth Largest Element (Quick Select)

Advanced
- Quick Sort Implementation

Remaining 10–20%
1. Heap / Priority Queue + Arrays

Used when you need the largest/smallest element repeatedly.

Problems:

Kth Largest Element
Top K Frequent Elements
Find Median from Data Stream
Merge K Sorted Arrays

Pattern:
import heapq
2. Monotonic Queue (Advanced Sliding Window)

Many people learn Sliding Window but skip this.

Problems:

Sliding Window Maximum
Constrained Subsequence Sum

Pattern:
from collections import deque
3. Sorting-Based Tricks

Sometimes sorting is the main idea.

Problems:

Merge Intervals
Meeting Rooms
Minimum Number of Arrows to Burst Balloons
Largest Number

Pattern:
