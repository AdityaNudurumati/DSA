==================================================
GRAPHS / 17. NETWORK FLOW
==================================================

When to think of this?
- maximum throughput source→sink, or problems reducible to it (matching, min cut)
- augmenting paths in a residual graph

Sub-patterns
│
├── Ford-Fulkerson     repeatedly find an augmenting path (DFS), push flow
├── Edmonds-Karp        Ford-Fulkerson with BFS augmenting paths -> O(VE^2)
├── Max Flow             value of the maximum source->sink flow (= min cut)
└── Bipartite Matching    model as flow with unit capacities

--------------------------------------------------

--------------------------------------------------
# residual capacity matrix cap[u][v]
# while BFS finds an augmenting path s->t with bottleneck b:
#     subtract b along the path, add b to reverse edges, max_flow += b
