--------------------------------------------------
Complexity cheat sheet
--------------------------------------------------
Bubble/Selection/Insertion : O(n^2)            (insertion is great for nearly-sorted)
Merge Sort                 : O(n log n), stable, O(n) space
Quick Sort                 : O(n log n) avg / O(n^2) worst, in-place, unstable
Heap Sort                  : O(n log n), in-place, unstable
Counting Sort              : O(n + k), stable    (k = value range)
Radix Sort                 : O(d (n + b))        (d digits, base b)
Bucket Sort                : O(n) average for uniform data

Comparison sorts have an Omega(n log n) lower bound; counting/radix/bucket beat it by
NOT comparing (they assume bounded keys).
