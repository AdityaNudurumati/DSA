==================================================
BIT MANIPULATION / 9. BITWISE TRIE
==================================================

Insert numbers' bits MSB->LSB into a binary trie; greedily walk toward the OPPOSITE
bit to maximize XOR (or the SAME bit to minimize).

Sub-patterns
│
├── Maximum XOR Pair       prefer opposite bit at each level
├── XOR Queries             max XOR with a value constraint (offline)
├── Minimum XOR Pair         prefer same bit (or sort + adjacent)
└── Offline XOR Queries       sort queries, insert eligible numbers as you go

--------------------------------------------------
Template
--------------------------------------------------
# see Trie section for the full implementation
# max_xor(num): walk root; want = 1 - bit; take `want` child if present (set that bit)
