==================================================
STRINGS / 6. STRING CONSTRUCTION
==================================================

When to think of this?
- Build or format an output string carefully (digits, layout, encoding)
- Parse input into a value; serialize a value into text

Sub-patterns
│
├── 1.string_builder/      accumulate chars in a list, ''.join at the end
│                          (digit-by-digit arithmetic on number strings)
├── 2.simulation/          follow formatting / layout rules step by step
└── 3.encoding_decoding/   serialize <-> parse (length-prefix, atoi, etc.)

--------------------------------------------------
Notes
--------------------------------------------------
- In Python, build with a list then "".join(parts) (strings are immutable).
- Grade-school arithmetic on digit strings: process from the least significant end,
  carry forward.
- Length-prefix encoding ("4#word") makes decoding unambiguous.
