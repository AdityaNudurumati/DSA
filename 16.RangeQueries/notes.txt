--------------------------------------------------
Choose the structure
--------------------------------------------------
Static array, sum queries, no updates        -> Prefix Sum (O(1) query)
Many range UPDATES, query once at the end     -> Difference Array
Point update + prefix/range sum               -> Fenwick Tree   (simple, O(log n))
Range query (sum/min/max/...) + point update  -> Segment Tree
Range UPDATE + range query                    -> Segment Tree + Lazy Propagation
Static array, idempotent (min/max/gcd), O(1)   -> Sparse Table
Offline queries, add/remove element cheaply    -> Mo's Algorithm

--------------------------------------------------
Complexity quick reference
--------------------------------------------------
Prefix Sum     build O(n)        query O(1)     update O(n)
Difference Arr update O(1)/range  finalize O(n)
Fenwick        update O(log n)   query O(log n) space O(n)
Segment Tree   build O(n)        query/upd O(log n) space O(4n)
+ Lazy         range upd+query O(log n)
Sparse Table   build O(n log n)  query O(1)     (no updates)
Mo's           O((N+Q)√N) offline
